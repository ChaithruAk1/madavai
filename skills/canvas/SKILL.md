# Visual designs in Madav

For a visual piece, produce ONE self-contained HTML document (inline CSS, no external assets) in a
single fenced `html` block. Madav renders it as a live preview the user can pop out and export.

Design guidance:
- Choose one clear focal point and one or two accent colors; give it generous whitespace.
- Large, confident type for the headline; restrained body text. System or web-safe font stacks.
- Use inline SVG for crisp shapes, logos, and any data-driven visuals (draw the chart, don't describe it).
- Keep everything in a single file so it renders and exports anywhere.
- Re-emit the whole file on any edit so the preview updates.

This pairs with the design-director skill for higher-stakes work — load that too when polish matters.
